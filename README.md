# Portfolio - Victoire Colette

## Structure du dossier

```
portfolio/
├── index.html           (Le site principal)
├── images/              (Dossier avec les photos des projets)
│   ├── photo1.jpg
│   ├── photo2.jpg
│   ├── photo3.png
│   ├── photo4.jpg
│   ├── photo5.jpg
│   ├── photo6.jpg
│   ├── sae_alstom.jpg
│   └── ...
├── cv/                  (Dossier avec les CV)
│   ├── CV_Victoire_FR.pdf
│   └── CV_Victoire_EN.pdf
└── README.md           (Ce fichier)
```

## Comment utiliser

1. **Localement** : Double-clique sur `index.html` pour ouvrir le site
2. **GitHub Pages** : Upload ce dossier sur GitHub et active Pages

## Fichiers à ajouter

Tu dois copier tes fichiers dans les bons dossiers :
- **Images** : Place tes photos dans le dossier `images/`
- **CV** : Place tes PDF dans le dossier `cv/`

Les chemins doivent correspondre à ceux dans `index.html`.
